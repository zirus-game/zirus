# Zirus: Infect. Solve. Conquer

This website is currently under active development since March 2026.

## Current Plan

For this website, it will be a game where the user is a computer virus. Solve puzzles and infect documents to 50% disk occupation. This will include many challenges:

- ensure the imaginary "User" of the infected computer does not discover/ you (the virus) and attempt to erase you.
- infect more and more files to gain dominance of the User's computer.
